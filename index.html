Averil

A Minimal CSS Framework

#Getting Started

Averil is small compilation of styled UI elements into a framework to help you build responsive apps and websites. It is fairly small in size 40KB (~8KB Compressed/GZip) It uses expressions classes to allow for easier use, so you only need to remember what you want to apply and put a prefix.

Expressive? Whats that?

In this case, it allows you to write the classes and modifiers in a way that everyone can understand.

Expressive prefixes used are as following:

    .is-*     // used to assign state or apply modifier
    .has-*  // used for child elements to components
    .can-*     // defines control classes to determine what elements 'can' do
    .do-*     // control modifier to handle actions

Installation

1. clone the repo ```git clone [link]```
2. import the averil.scss to your main stylesheet using ```@import path/to/averil```
3. You are good to go

#Elements

Content

Default styling for standard text content, no classes needed

Default HTML tags for the content have been styled for use out of the box without a need for any additional classes. Following HTML tags are handled by default:

  • <p> paragraphs
  • <ul>, <ol>, <dl> lists
  • <h1> to <h6> headings
  • <blockquotes> quotes
  • <em> and <strong>
